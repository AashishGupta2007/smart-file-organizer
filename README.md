# 🚀 Smart File Organizer CLI

A powerful Python-based Command Line Tool that automatically organizes
files into categorized folders with preview, logging, and undo support.

------------------------------------------------------------------------

## ✨ Features

✅ Automatic file organization\
✅ Preview mode (safe execution)\
✅ Undo last operation\
✅ Colored terminal output\
✅ Activity logging\
✅ Modular architecture

------------------------------------------------------------------------

## 📂 Supported Categories

-   Images
-   Documents
-   Videos
-   Music
-   Archives
-   Others

------------------------------------------------------------------------

## ⚙️ Installation

Clone repository:

``` bash
git clone https://github.com/AashishGupta2007/smart-file-organizer.git
cd smart-file-organizer
```

Install dependencies:

``` bash
pip install -r requirements.txt
```

------------------------------------------------------------------------

## 🚀 Usage

### Preview changes

``` bash
python main.py --path ~/Downloads --preview
```

### Organize files

``` bash
python main.py --path ~/Downloads
```

### Undo last organization

``` bash
python main.py --undo
```
